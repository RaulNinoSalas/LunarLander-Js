ACERCA DE LUNAR LANDER

Autor: Raúl Niño Salas
Creación: Diciembre de 2017
Descripción: Aterrizar suavemente la nave sobre la superficie.

HERRAMIENTAS DE DESARROLLO

El proyecto ha sido desarrollado con código html, css, y javaScript.
El diseño del proyecto tiene dos versiones: una versión responsive, para
pantallas de ordenador horizontales, y una versión smartphone, optimizada para pantalla vertical.
El juego ha sido diseñado consultando los siguientes enlaces:
  • Para el tratamiento del código html y css: http://www.w3schools.com/html/
  • Para tratar imagenes: https://pixlr.com/
  • Para organizar el proyecto: https://github.com/RaulNinoSalas/LunarLander-Js
  • Para ayudas de pequeñas de dudas: https://www.google.es/
